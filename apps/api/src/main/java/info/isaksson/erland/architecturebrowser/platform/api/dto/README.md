# API DTOs

DTOs are grouped by response area instead of one large snapshot aggregation file.

## Snapshot DTO groupings
- `SnapshotImportDtos` — import responses
- `SnapshotResponseDtos` — summary, detail, overview responses
- `SnapshotPayloadDtos` — full payload and payload item records
- `SnapshotSharedDtos` — shared snapshot support records

`SnapshotDtos` remains only as a deprecated migration marker and should not receive new DTOs.
